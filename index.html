    • Atividade TABULEIRO

    Resumo da Atividade (realizada em grupo)

    Os jogos de tabuleiro são jogos que utilizam superfícies, normalmente planas, pré-determinadas, com marcações de acordo com as regras de cada jogo. Desenvolvidos para serem jogados por um ou mais participantes, os jogos de tabuleiro podem demandar estratégia, sorte, conhecimento ou memória para o desenrolar do jogo.

    Para codificar o Jogo foi nécessario usar o for e while que são estruturas de repetição, também conhecidas como loops, são utilizadas para executar repetidamente uma instrução ou bloco de instrução enquanto determinada condição estiver sendo satisfeita.

    Assim colocando as regras criadas e um dado online para conseguir avançar as casas.

    Resumo da Atividade

    Vetores são estruturas de dados que armazenam usualmente uma quantidade fixa de dados de um certo tipo; por esta razão, também são conhecidos como estruturas homogêneas de dados.

    Na programação apresentamos um vetor para receber 5 números e ao final ele fazer um conta de soma de todos os números apresentados
